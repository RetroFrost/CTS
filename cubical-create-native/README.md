# Cubical Create native build

Native migration preview for Cubical Create.

- Linux frontend: GTK 4
- Windows frontend: Win32 Common Controls and Windows Shell file dialogs
- Shared core: C++20

The generated Linux package is an amd64 `.deb`. The Windows artifact is a portable x64 `.exe` ZIP.

This is the compiled native shell preview. The full CTS-compatible renderer and exporter remain in migration.
